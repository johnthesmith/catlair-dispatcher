Intro

Прототип сервиса dispatcher выполняет задачу диспетчеризации вызовов на множество исполнителей.

API

dispatcher/collect-payloads

Сбор информациции об исполнителях для диспетчера на основе которой он будет выполнять диспетчеризацию.

dispatcher/get-payloads

Возвращает собранную посредством dispatcher/collect-payloads информацию о перечне полезных нагрузок участников.

dispatcher/build-stat

Строит и возвращет статистику по выполненынм исполнителями полезным нагрузкам.

dispatcher/clear-stat

Очищает статистику по выполнениям полезных нагрузок исполнителями.
